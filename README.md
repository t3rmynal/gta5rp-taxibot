# 🚕 GTA5RP TaxiBot

> [!NOTE]
> **Репозиторий архивирован - обновлений больше не будет.**
> Программа может работать, однако актуальность не гарантируется.

**GTA5RP TaxiBot** - помощник для таксистов на **GTA5RP**.
Автоматически отслеживает входящие вызовы в КПК такси и мгновенно их принимает.

---

## ⚙️ Как использовать

1. Запустите программу
2. Откройте КПК такси
3. Включите поиск вызовов
4. Ожидайте (при принятии вызова проиграется звук)
